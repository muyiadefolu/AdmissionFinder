# AdmissionFinder

ASP.NET Core MVC application for the AdmissionFinder contribution dashboard pages.

## Development

Requirements:

- .NET 8 SDK

Run locally:

```sh
dotnet run
```

Then open the URL shown in the terminal. The main routes are:

- `/`
- `/compare`
- `/admin-pulse`

## Project Structure

- `Controllers/` contains MVC controllers.
- `Views/` contains Razor views.
- `public/` contains static assets such as the favicon.
